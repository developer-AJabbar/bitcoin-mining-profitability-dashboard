## 🔍 Problem Statement

Bitcoin miners operate in a highly volatile environment where revenue, costs, and network conditions change daily. Without clear analytics, miners cannot answer:

- Is my operation profitable today — and why?
- At what BTC price do I break even?
- How does network difficulty affect my earnings?
- What is my ROI, and how does electricity cost impact it?

This dashboard answers all of these questions using real market data and a realistic miner fleet model.

## 🚀 Quick Start

### Step 1 — Install dependencies
```bash
pip install requests pandas numpy
```

### Step 2 — Collect all data
```bash
python collect_data.py
```
This fetches BTC price history from CoinGecko, network hashrate/difficulty from Blockchain.com, and generates a realistic 135-unit miner fleet. Takes ~30 seconds.

### Step 3 — Import into Power BI
1. Open **Power BI Desktop**
2. Home → **Get Data** → Text/CSV
3. Select `data/master_mining_data.csv`
4. Click **Load**
5. Optional: also import `miner_fleet.csv` as a second table

### Step 4 — Add DAX measures
Open `dax_measures.dax` — copy each measure into Power BI via **Modeling → New Measure**.

## 📊 Dataset Columns (master_mining_data.csv)

| Column | Description |
|--------|-------------|
| `date` | Date (YYYY-MM-DD) |
| `btc_price_usd` | BTC closing price |

| `gross_profit_usd` | Revenue minus total cost |

## 📋 Power BI Dashboard Pages

### Page 1 — Executive Summary
**Visuals:**
- KPI cards: Total Revenue · Total Profit · Margin % · Profitable Days %
- Line chart: Daily Revenue vs Daily Cost (2-year)
- Gauge: Current BTC Price vs Break-even Price
- Card: Total BTC Mined

**Slicers:** Year · Quarter

---

### Page 2 — Profitability Analysis
**Visuals:**
- Area chart: Gross Profit over time
- Combo chart: BTC Price (line) + Break-even Price (line) + Profit margin (bar)
- Scatter plot: BTC Price vs Gross Profit (reveals correlation)
- KPI: Break-even gap (how far above/below break-even)
- Conditional formatting table: Monthly profit with color coding

---

### Page 3 — Cost Analysis
**Visuals:**
- Donut chart: Cost breakdown (Electricity / Pool Fee / Other OPEX)
- Bar chart: Monthly electricity cost
- Line chart: Revenue per kWh over time (efficiency trend)
- KPI cards: Cost per BTC · Avg kWh/day · Avg $/kWh
- Waterfall chart: Revenue → costs → profit

---

### Page 4 — Forecasting
**Visuals:**
- Line chart: 30-day moving average (Revenue + Profit — smooths BTC volatility)
- YoY comparison bar chart
- Forecast line (Power BI built-in analytics pane → Forecast)
- Trend annotations: difficulty adjustments, halving event

---

### Page 5 — What-If Analysis
**Setup:** Create 3 What-If Parameters via Modeling → New Parameter:
- **BTC Price Scenario**: $10,000 → $150,000, step $5,000
- **Electricity Cost**: $0.02 → $0.12, step $0.005
- **Hashrate Scale**: 0.5× → 3.0×, step 0.1

**Visuals:**
- Scenario Profit KPI card (updates live with sliders)
- Scenario Margin % gauge
- Break-even BTC price output card
- Combo chart: Scenario Revenue vs Scenario Cost

This is the most impressive page for clients — interactive sliders that show real-time profit changes.

## 💡 Key Insights This Dashboard Reveals

- **Break-even analysis**: shows exact BTC price where mining turns unprofitable
- **Difficulty impact**: as network hashrate grows, individual miner earnings compress even if BTC price stays flat
- **Electricity dominance**: electricity typically represents 65–75% of total mining cost — the single biggest lever for profitability
- **Halving effect**: block reward halving events cause immediate revenue compression, visible in the trend chart
- **ROI seasonality**: mining ROI peaks during BTC bull runs and compresses during bear markets
